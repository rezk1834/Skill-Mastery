# Skill Mastery Tracker

Skill Mastery Tracker is a Flutter application that allows users to track their skill mastery progress. Users can add skills, record time spent on each skill, set start dates, and visualize their progress.

## Features

- **Add Skills**: Add new skills with names, colors, start dates, and optional notes.
- **Track Progress**: Record time spent on each skill and visualize progress with a percentage indicator.
- **Edit and Delete Skills**: Modify existing skills or remove them as needed.


## Installation

To run the Skill Mastery Tracker app locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/rezk1834/Skill-Mastery.git
   ```
   
2. Navigate to the project directory:
   ```bash
   cd Skill-Mastery
   ```

3. Install dependencies:
   ```bash
   flutter pub get
   ```

4. Run the app:
   ```bash
   flutter run
   ```
